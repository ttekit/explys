# Explys — SPA

React + Vite client for **Explys**. Full setup, backend workflows, repo structure, environment variables, and **monthly Gemini token budgeting** live in the [root README](../README.md).

Quick start:

```bash
npm ci
cp .env.example .env   # configure API URL / tokens if required
npm run dev
```

Build: `npm run build`. Preview production build: `npm run preview`.
